# Ejemplos de TypeScript para Clases

Este repositorio contiene ejemplos prácticos de TypeScript organizados por temas. Cada carpeta incluye un conjunto de archivos que ilustran conceptos clave del lenguaje.

## Estructura del Proyecto

- **01-introduccion/**: Conceptos básicos de TypeScript.
- **02-tipos-basicos/**: Tipos primitivos, arrays, y más.
- **03-funciones/**: Declaración y uso de funciones.
- **04-tipos-avanzados/**: Alias, interfaces, y tipos avanzados.
- **05-genericos-utility/**: Uso de genéricos y tipos utilitarios.
- **06-clases-poo/**: Programación orientada a objetos con clases.
- **07-modulos/**: Uso de módulos en TypeScript.
- **08-di-solid/**: Principios SOLID y inyección de dependencias.
- **09-decoradores/**: Uso de decoradores en TypeScript.
- **10-comparacion-js-ts/**: Comparación entre JavaScript y TypeScript.
- **11-seguridad-validacion/**: Validaciones y seguridad en TypeScript.

## Cómo Usar Este Repositorio

1. Clona el repositorio:
   ```bash
   git clone https://github.com/MercadoFabio/typescript-clase-ejemplos.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Explora las carpetas y ejecuta los ejemplos con:
   ```bash
   npx tsc && node <archivo>
   ```
